# GENO Steps

Move more. Compete with friends. Stay connected to GENO Wellness.

## Download

[Download GENO Steps for Android](https://github.com/GENO-Wellness/genosteps-download/raw/main/downloads/GENO-Steps-v1.0.10.apk)

If your phone asks for permission to install the app, allow installation from your browser or file manager, then continue.

- Track your daily steps.
- Join private step competitions.
- Create invite-only challenges.
- View daily rankings.
- Sync your progress with GENO Wellness.
